Kid Bingo

This side-project was developed as a way to gamify my time watching my kid play with friends. The webapp uses React, BrowserRouter, and Xano.

Live Link

Project Overview

The reason for this project was to make the dreadfully boring task of watching my kid play with his friends more bareable. After noticing that there were some common and uncommon occurances while playtime happened, I took note of what was happening and decided to make a silly Bingo game.

I started by creating some React components to handle the basic structure of the bingo board and used some dummy data to mock the squares up. Once the board was built and rendering correctly, I went to Xano to host my own API to handle the bingo terms. Built some API fetch requests and used React.useEffect to take care of the fetch requests. BrowserRouter is used for adding the second page and handling the navigation layout.

After the data was coming in from the API, I worked on making the squares clickable and lastly I added win conditions. For the win condition, I had to create an array of winning combinations which were arrays. Every time there are five or more boxes checks, the game runs some code to determine if there is a matching combination.

For the victory message, I have an array of somewhat insulting victory messages. Because in order to win this game, you probably lost a little bit of your sanity. Finally, this project briefly started in Scrimba and then was exported into GitHub where I handled all the updates using GitHub Desktop and VS Code Studio. The final project is hosted on Netlify.
